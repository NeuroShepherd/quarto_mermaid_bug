# Quarto Mermaid*Tabset Panel Bug

This repo is a minimal reprex for what appears to be a bug (or at least unexpected behavior) when trying to render a mermaid diagram within a tabset-panel in Quarto. To see this locally, run the following code:

```bash
git clone git@github.com:NeuroShepherd/quarto_mermaid_bug.git
cd quarto_mermaid_bug
quarto preview
```

Or take a look at the pre-rendered site: https://neuroshepherd.github.io/quarto_mermaid_bug/

Rendered with Quarto version: 1.8.24
